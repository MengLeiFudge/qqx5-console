# qqx5-console

qq炫舞手游爆气表（纯控制台），包含星弹泡弦bytes文件转化及apk提取功能

注意，现在本程序已经无法计算排位歌曲爆点，只能计算自制爆点。

## 程序说明

程序主要功能为根据谱面文件计算得到爆点。

目前版本已经修改过多次，较为完善，一分钟就可以算完两千首歌曲，算法其实没必要过于优化。

耗费时间占比：读取信息10%，计算单排爆点10%，计算双排爆点70%，爆点信息写入文件10%。
